# WAD

A WAD file (Where's All the Data) is the game data archive for DOOM, containing all levels, graphics, sounds, and textures. The doomgeneric engine does not include it. 
- Classic free shareware version [DOOM1.wad](https://doomwiki.org/wiki/DOOM1.WAD). About 4MB.  
- Smaller size - [Reworked Shareware IWAD for embedded devices](https://github.com/fragglet/squashware/tree/trunk).  
- Extremely small - [Minimal Doom IWAD](https://github.com/fragglet/miniwad).
